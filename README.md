# my-agent

A local AI agent that reads files using the MCP protocol. No cloud services involved.

## Setup

ollama pull qwen2.5:3b
pip install -r requirements.txt
python client.py

## Expected output

MCP server connected.
AI wants to use tool: read_file
File read successfully
Agent: Here are the key points:
1. CAN-FD network at 500 kbps
2. Logging in WARNING mode
3. Fail-safe mode set to SHUTDOWN
